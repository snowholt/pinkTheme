# Soft Pink Theme for Visual Studio Code

## Overview
The Soft Pink Theme is a beautifully crafted theme designed for a feminine aesthetic, providing a soft and calming color palette for your coding environment. This theme enhances your coding experience with its gentle hues and elegant styling.

## Features
- Soft pink color scheme that is easy on the eyes
- Customizable settings to adjust the theme to your preference
- Designed for a pleasant coding experience

## Installation
To install the Soft Pink Theme, follow these steps:

1. Open the Extensions view in your editor (you can do this by clicking on the Extensions icon in the Activity Bar on the side of the window).
2. Search for "Soft Pink Theme".
3. Click on the Install button.



## Usage
Once installed, you can activate the Soft Pink Theme by following these steps:

1. Open the Command Palette (Ctrl+Shift+P or Cmd+Shift+P on macOS).
2. Type "Color Theme" and select "Preferences: Color Theme".
3. Choose "Soft Pink Theme" from the list.
